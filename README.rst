Install
------------------------------------------------------------------------------

``lbdrabbit`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install lbdrabbit

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade lbdrabbit
